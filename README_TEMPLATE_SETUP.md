# New Project Template Setup

## 1) Copy this folder
- Copy `_ProjectTemplate` to a new folder at Vault root.
- Rename it to your project folder name (example: `MY-PROJECT`).

## 2) Replace placeholders
- Replace all `PROJECT` with your project name.
- Replace all `project/project-slug` with your tag slug (example: `project/my-project`).

## 3) Start with these files
- `notes/_index.md`
- `DECISIONS.md`
- `CHANGELOG_DEV.md`
- `TROUBLESHOOTING.md`
- `.cursor/rules/obsidian-doc-sync.mdc`

## 4) First run checklist
- Open `notes/_index.md` and confirm links.
- Add at least one note and include `parent` + `tags`.
- Add today's entry to `CHANGELOG_DEV.md`.
