# マニュアル用スクリーンショット

`現場向け操作マニュアル.md` から参照する画像を、このフォルダに保存してください。

## 撮影のコツ

- **スマートフォン実機**で撮影する場合: 縦向きで全体が入るようにする
- **PC**の場合: ブラウザのウィンドウを適度な幅にし、**DevTools は閉じた状態**で撮る
- 個人情報・在庫の機密値が写り込まないよう、**テスト用データ**または**モザイク**を検討してください

## 推奨ファイル名一覧

| ファイル名 | 内容 |
|------------|------|
| `01-pin.png` | 起動直後の暗証番号（PIN）入力画面 |
| `02-portal.png` | ポータル（4つの業務カードが並んだ画面） |
| `03-inventory-start.png` | 棚卸し「棚卸しを開始」：店舗・月・実施日の選択 |
| `04-inventory-confirm-store.png` | 棚卸し「店舗の確認」モーダル |
| `05-inventory-tupper.png` | 棚卸し Step1 タッパー入力（店舗の場合） |
| `06-inventory-stock.png` | 棚卸し 在庫（フレーバー）入力 |
| `07-inventory-review.png` | 棚卸し 入力内容確認（Step4） |
| `08-inventory-footer.png` | 下部の「戻る」「次へ」「完了する」ボタン例 |
| `09-request-start.png` | 補充依頼「補充依頼を開始」 |
| `10-request-board.png` | 補充依頼 依頼ボード（数量入力中） |
| `11-transfer-top.png` | 移動記録 トップ（対象月・実施タスク選択） |
| `12-transfer-issue.png` | 移動記録 起票の入力または確認の一例 |
| `13-stock-month.png` | 在庫量確認 対象月選択と一覧 |

画像を配置したら、リポジトリにコミットするか、社内共有フォルダにマニュアルと一緒に置いてください。
